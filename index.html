Tentang Saya

Halo! Nama saya Wiwin Purwanti. Saya adalah anak terakhir dari dua bersaudara dan saat ini berusia 16 tahun. Saya sangat menyukai musik dan sering menghabiskan waktu dengan mendengarkan berbagai genre musik yang saya suka. Warna favorit saya adalah pink, dan saya juga sangat menyukai coklat sebagai makanan! Di samping hobi dan kesukaan saya, saya selalu bersemangat untuk mengeksplorasi hal-hal baru dan mempelajari keterampilan baru. Saat ini, saya aktif dalam Ekstrakulikuler Pramuka dan Pmr. Saya percaya bahwa hobi dan minat saya memainkan peran penting dalam membentuk kreativitas dan kepribadian saya. Terima kasih telah mengunjungi halaman saya! Saya berharap dapat berbagi lebih banyak tentang perjalanan saya dan pencapaian saya di sini.
